# Pong Game in C++

A small Pong game built during C++ coursework to practice object-oriented programming, game loops, collision handling, scoring, and simple AI behavior.

[![Watch the demo](https://img.youtube.com/vi/dsDNGdwUADs/0.jpg)](https://youtu.be/dsDNGdwUADs)

## Features

- Player-controlled paddle
- AI-controlled opponent
- Ball movement and paddle collision detection
- Score tracking
- Round reset after each point
- Simple game loop using Raylib

## Built With

- C++
- Raylib

## How to Run on Windows

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open the integrated terminal with **Terminal > New Terminal**.
4. Move to the folder that contains `main.exe`.

Example:

```powershell
cd "D:\Pong-main\Pong-main"
```

Replace the path above with the actual location of the project on your computer.

5. Run the game:

```powershell
.\main.exe
```

## Notes

- `main.exe` must be in the current terminal directory.
- The included executable is intended for Windows.
- If the game does not start, check that the terminal is opened in the correct folder.
- Visual Studio Code is only used to open the folder and run the executable; no additional setup is required when `main.exe` is included.
