PONG GAME (C++ / RAYLIB)

A simple Pong game developed while learning C++.
The project includes basic game logic, collision handling, scoring, and an AI-controlled opponent.

DEMO
https://youtu.be/dsDNGdwUADs

HOW TO RUN ON WINDOWS

1. Download or clone the repository to your computer.

2. Open Visual Studio Code.

3. Open the project folder in Visual Studio Code.

4. Open the integrated terminal:
   Terminal > New Terminal

5. Move to the folder that contains main.exe.
   Example:

   cd "D:\Pong-main\Pong-main"

   Replace the path above with the actual location where the project is stored on your computer.

6. Run the game with:

   .\main.exe

NOTES

- main.exe must be located in the current terminal folder.
- If the game does not start, check that the terminal path is correct.
- This executable is intended for Windows.
- Visual Studio Code is only used to open the folder and run the command; no additional setup is required if main.exe is already included.
